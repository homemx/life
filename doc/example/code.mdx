## react live 例子

### 可编辑

```jsx live
() => {
  const [count, setCount] = React.useState(1)

  return (
    <>
      <p>{count}</p>
      <Button type="primary" onClick={() => setCount(count + 1)}>increase</Button>
    </>
  )
}
```

### 可编辑 2，注入 component
```jsx live version="1.2.0"
() => {
  return (
    <>
      <Button type="primary" onClick={
        () => Modal.success({
          title: 'hello',
          content: 'world'
        })
      }>modal</Button>
    </>
  )
}
```


### 直接render
```jsx render
() => {
  const [count, setCount] = React.useState(1)

  return (
    <>
      <p>{count}</p>
      <Button type="primary" onClick={() => setCount(count + 1)}>increase</Button>
    </>
  )
}
```
